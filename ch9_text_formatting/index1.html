HTML
HyperText Markup Language, the standard language used to create and structure web pages.
CSS
Cascading Style Sheets, used to style and design web pages.
JavaScript
A programming language that makes websites interactive and dynamic.
DOM
Document Object Model, a programming interface that represents an HTML document.
Semantic HTML
HTML elements that describe the meaning of their content, such as <header> and <article>.
Responsive Design
A web design approach that adapts a website to different screen sizes and devices.
API
Application Programming Interface, which allows different software applications to communicate.
HTTP
HyperText Transfer Protocol, used for transmitting web pages over the internet.
URL
Uniform Resource Locator, the web address used to locate resources on the internet.
SEO
Search Engine Optimization, the practice of improving a website's visibility in search engine results.
